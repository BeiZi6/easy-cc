# 09 版本更新日志

> Claude Code 大约每周都有更新。这里记录**对使用方式有影响**的关键变更。
> 不是 changelog 镜像——只挑会改变你日常用法的事。

## 维护规则

每条记录三件事：
1. **日期 + 版本号**（如已知）
2. **变更点 + 原因**（猜不准就只写"变化"）
3. **受影响章节**：用 wiki link 指过去，并去那个章节顶部加 `> [!warning] 自 vX 起 ...` 提示

模板：

```markdown
## YYYY-MM-DD · vX.Y.Z

- 新增：xxx
- 变更：xxx
- 弃用：xxx
- 受影响章节：[[../04-配置与定制/04-Hooks钩子系统]]
```
