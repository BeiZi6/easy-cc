# Slash 命令一览

> 在输入框开头打 `/`，回车即可。命令集会随版本变化，本表注明发现日期。

## 内置命令（按用途分组）

### 会话控制

| 命令 | 作用 |
|------|------|
| `/clear` | 清空当前上下文（不退出登录） |
| `/compact` | 压缩上下文（保留要点，丢冗余） |
| `/resume` | 恢复上次会话 |
| `/exit` | 退出 |
| `/help` | 帮助 |

### 模式切换

| 命令 | 作用 |
|------|------|
| `/plan` | 进入 Plan 模式（不动手只列计划） |
| `/auto` | 切换自动放行 |
| `/fast` | 切换 Fast 模式（更快但策略简化） |

### 设置 / 信息

| 命令 | 作用 |
|------|------|
| `/config` | 打开配置 UI |
| `/model` | 切换模型（Opus / Sonnet / Haiku） |
| `/cost` | 查看本会话花了多少 token / 钱 |
| `/status` | 当前模型 / 模式 / 目录 / 权限 |

### 项目相关

| 命令                 | 作用                    |
| ------------------ | --------------------- |
| `/init`            | 生成 CLAUDE.md（初始化项目说明） |
| `/review`          | 审查当前 PR               |
| `/security-review` | 安全视角审查                |

### 工具相关

| 命令 | 作用 |
|------|------|
| `/agents` | 管理 subagent |
| `/mcp` | 管理 MCP 服务器 |
| `/hooks` | 管理 hooks |
| `/permissions` | 编辑权限 |
| `/memory` | 编辑记忆文件 |

> [!warning] 命令集随版本更新
> 不同版本会增删命令。建议在装新版本后跑 `/help` 看一眼现有命令再用本表对照。

## 自定义 Slash 命令
