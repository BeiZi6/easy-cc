# 快捷键与命令速查

> 半屏可贴。命令集随版本变化，对不上就跑 `/help` 校准。

## 输入框前缀

| 前缀 | 作用 |
|------|------|
| 普通 | 给 Claude 发消息 |
| `/` | slash 命令 |
| `!` | 直接执行 shell（输出进上下文） |
| `#` | 写入 memory |
| `@` | 引用文件 / 目录 / agent / MCP resource |

## 必备 slash 命令

| 命令 | 作用 |
|------|------|
| `/help` | 查看所有命令 |
| `/clear` | 清空上下文 |
| `/compact [focus]` | 压缩上下文（可指定保留什么） |
| `/rewind` | 回到 checkpoint |
| `/btw` | 侧问，不入历史 |
| `/plan` | 进 Plan 模式 |
| `/auto` | 切 Auto 模式 |
| `/fast` | 切 Fast 模式 |
| `/model` | 换模型 |
| `/cost` | 看花了多少 |
| `/status` | 当前状态 |
| `/init` | 生成 CLAUDE.md |
| `/review` | 审 PR |
| `/security-review` | 安全审 |
| `/agents` | 管 subagent |
| `/mcp` | 管 MCP |
| `/hooks` | 管 hooks |
| `/permissions` | 改权限 |
| `/memory` | 编辑记忆 |
| `/config` | 配置 UI |
| `/schedule` | 创建 routine |
| `/loop` | 会话内定时 |
| `/plugin` | 浏览 plugin marketplace |
| `/exit` | 退出 |

## 常用键盘快捷键

| 按键 | 作用 |
|------|------|
| `Esc` | 中断当前动作 |
| `Esc Esc` | 打开 rewind 选项 |
| `Ctrl+C` | 取消当前输入 |
| `Ctrl+D` | 退出 |
| `↑` / `↓` | 翻历史输入 |

## CLI 启动 flag

| flag | 作用 |
|------|------|
| `-p "..."` | 非交互（headless） |
| `--output-format json\|stream-json\|text` | 输出格式 |
| `--allowedTools T1,T2` | 工具白名单 |
| `--permission-mode <mode>` | 启动模式 |
| `--continue` | 续上次 session |
| `--resume <id>` | 指定 session 续上 |
| `--worktree <name>` | 在隔离 worktree 跑 |
| `--from-pr <num>` | 从 PR 开 session |
| `--model <id>` | 指定模型 |
| `--version` | 看版本 |

## 常用一行命令

```bash
# 装 / 升级
npm install -g @anthropic-ai/claude-code
claude --version

# 装 GitHub App
# （在仓库里跑：）/install-github-app

# 加 MCP
claude mcp add <name>
claude mcp list
claude mcp logs <name>

# headless 跑一次
claude -p "审查当前 diff" --allowedTools Read,Grep,Bash --output-format json

# 起一个 worktree
claude --worktree feature-x
```

## 自定义 slash 文件位置速查

```
~/.claude/commands/        # 全局命令
.claude/commands/          # 项目命令（入 git）
.claude/commands.local/    # 项目个人（不入 git）
.claude/agents/            # subagent 配置
.claude/skills/            # skill 包
.claude/hooks/             # hook 脚本
.claude/output-styles/     # 自定义输出风格
.claude/settings.json      # 项目设置
.claude/settings.local.json # 项目个人设置（不入 git）
~/.claude/settings.json    # 全局设置
~/.claude/CLAUDE.md        # 全局项目说明书
CLAUDE.md                  # 项目说明书（入 git）
CLAUDE.local.md            # 项目个人说明书（不入 git）
```
