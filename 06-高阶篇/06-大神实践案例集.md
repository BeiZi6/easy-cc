# 大神实践案例集

> 收录公开来源的"出圈"实践，每条带链接 + 一句话核心。
> 立场不同流派各异，并列在此供你自取所需。

## Anthropic 官方

### Claude Code Best Practices（必读）
- 链接：https://code.claude.com/docs/en/best-practices
- 核心：**Context 是稀缺资源**；Explore → Plan → Code → Commit 四段式；让 Claude 有"自验证"的手段；用 subagent / plan / sandbox 三种方式降打扰。

### Common Workflows（recipe 集）
- 链接：https://code.claude.com/docs/en/common-workflows
- 核心：bug fix / refactor / 写测试 / 打 PR / `/loop` 定时——每个有最小 prompt 模板。

### April 23 Postmortem（2026/04/24）
- 链接：https://www.anthropic.com/engineering/april-23-postmortem
- 核心：三个 harness bug 让用户以为模型变差了。其中"久 idle 后变健忘"非常经典——不是模型问题，是 harness。
- 启示：抱怨"模型变笨"前，先看版本号。

## Geoffrey Huntley（极简派）

### everything is a ralph loop
- 链接：https://ghuntley.com/loop/
- 核心：**反对**复杂多 agent。一个 while-loop 跑同一个 Claude，让它读 spec 干活、干完再读再干。正向（building）+ 反向（clean-room 验证）配合。

### I ran Claude in a loop for three months
- 链接：https://ghuntley.com/cursed/
- 核心：长期 ralph loop 实战，做出了 cursed 编程语言。金句：「LLMs are mirrors of operator skill」。

### Porting is trivial
- 链接：https://ghuntley.com/porting/
- 核心：Claude Code 跨语言移植代码极其便宜，只要测试齐。

### The Final Bottleneck（2026/02/13）

### The Center Has a Bias（2026/04/11）
